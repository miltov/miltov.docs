overview

This documentation describes all classes, methods, and keywords used by the .miltov interpreter and its modules: robloxbot, webbackend, and miltovinterpreter.

The robloxbot module interfaces with Roblox web APIs to automate Roblox-related tasks.

The webbackend module provides a minimal Express.js backend for web integrations.

The miltovinterpreter parses and executes .miltov scripts and offers language constructs and standard library functions.

robloxbot

The robloxbot class wraps Roblox web API calls using your Roblox security cookie.

constructor(cookie)
Creates a new robloxbot instance with the provided Roblox security cookie (used for authenticated requests).
async getUserId()
Fetches your Roblox user ID from the /mobileapi/userinfo endpoint.
async getBalance()
Retrieves your Robux balance from the /economy/balance endpoint.
async getUserProfile(userId)
Returns Roblox profile info for a given userId.
async getPlayerGroups(userId)
Returns an array of groups the user belongs to, with their roles.
async setRank(groupId, userId, rankId)
Sets a user’s rank in a Roblox group.
async sendFriendRequest(userId)
Sends a friend request to the user ID.
async acceptFriendRequest(userId)
Accepts a pending friend request.
async declineFriendRequest(userId)
Declines a pending friend request.
async getFriends(userId)
Returns a list of the user’s friends.
async buyAsset(assetId)
Attempts to purchase an asset using its product ID.
async postShout(groupId, message)
Posts a shout in a specified group.
async joinGroup(groupId)
Joins the given group ID.
async leaveGroup(groupId)
Leaves the group, based on current user ID.
robloxbot = new robloxbot("your_cookie_here")
userId = robloxbot.getUserId()
balance = robloxbot.getBalance()
profile = robloxbot.getUserProfile(userId)
groups = robloxbot.getPlayerGroups(userId)
robloxbot.sendFriendRequest(userId)
robloxbot.acceptFriendRequest(userId)
robloxbot.setRank(123456, userId, 255)
webbackend

The webbackend class provides minimal HTTP backend features to serve your site or API.

constructor()
Creates a new backend server instance using Express.js.
start(port)
Starts the server on the specified port. Defaults to 3000 if not provided.
backend = new webbackend()
backend.start(8080)
keywords

The keywords are responsible for parsing and evaluating .miltov scripts using a custom syntax tree and evaluation engine.

milt x = "hello"
Declares a variable using the milt keyword.
print("text")
Outputs a value to the console.
func greet(name) { ... }
Declares a function.
if (...) { ... } else { ... }
Conditional execution.
while (...) { ... }
Looping structure.
message("text")
Sends a message using custom interpreter logic.
shout("text")
Posts a shout using Roblox integration.
milt username = "builderman"
print("Welcome", username)

func greet(name) {
  message("Hello " + name)
}

greet(username)
